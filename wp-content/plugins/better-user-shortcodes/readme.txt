Forgot wp-login.php, you can add the following shortcodes into your site: login, logout, register, forgot password, add custom fields to register.

== Description ==

Set up your custom registration form, add shortcodes into your posts, pages or sidebars and forget wp-login.php.

= Features =
* Login shortcode
* Logout shortcode
* Registration shortcode with lot of options
* Password reminder shortcode

== Installation ==

Unzip the plugin and copy the better-user-shortcodes folder to your wp-content/plugins folder or install from wordpress.org repository and activate the plugin.

== Frequently Asked Questions ==

= Can I add custom fields to registration? =

Yes, you can, you find instructions on the plugin's settings page.
